Closetbox


Es un sistema de almacenamiento de objetos y/o pertenencias que se encarga de recojer, trasladar, almacenar y retornar dichos elementos segun el tiempo deseado. Es un sistema que cuenta con diversos servicios como politicas de descuentos, pagos en linea, entre otros. Cuenta con las siguientes tecnologias: - Mongodb - Pivotal.io - BrainTree
